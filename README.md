<h1>Pasos para instalar el proyecto</h1>
<ul>
  <li>Clona el repositorio : git clone <tu-repo></li>
    <li>Entra a la carpeta backen: Entrar al backend</li>
    <li>Instala las dependencias npm install</li>
    <li>Inicia el servidor: npm start</li>
</ul>
